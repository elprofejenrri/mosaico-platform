# Product Notes

MOSAICO is a full-stack site for private online Spanish lessons.

Core systems:

- Public marketing pages and pricing
- Supabase Auth with Google OAuth
- FastAPI backend with Supabase Postgres
- Supabase Storage for CMS-uploaded images
- Stripe Checkout for paid bookings
- Admin CMS for products, availability, teachers, blog posts, users, and settings
- Optional Google Calendar integration for booking events

Deployment target: Render.
