# Coordinator Guide

## Purpose

Coordinators manage operational health: scheduling, teacher utilization, student needs, class status, availability, and support workflows.

## Responsibilities

- Monitor empty slots and booking pressure.
- Help teachers maintain availability.
- Support students and tutors.
- Review cancellations, no-shows, and reschedules.
- Keep class and feedback records complete.

## Permissions

Coordinator permissions are scoped below Admin and Super Admin. They should not manage platform settings, system roles, or protected permissions.

## Recommended Dashboard

Coordinator dashboard should focus on today/tomorrow operations, unresolved conflicts, empty slots, upcoming classes, student risk signals, and teacher utilization.
